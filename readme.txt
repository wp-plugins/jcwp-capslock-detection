This plugin shows a tooltip when user's have their CAPS lock on while typing their password to login.

== Description ==

This plugin gives you an options to make all password fields on your blog to show a tooltip message if user keyboard's CAPS lock is on.
You can enable or disable this plugin just by setting 'Enabled' under Settings > JCWP Scroll To Top > Enabled to "No"

== Installation ==

1. Upload unzipped plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Bot detector Options on left configure your preferences.
